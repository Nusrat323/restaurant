Restaurant React Template

This is a 6-page React frontend website template for restaurants. It is built with ReactJS, Vite, Tailwind CSS, Framer Motion, React Router DOM, Heroicons, and React Icons.

The template is frontend-only, fully responsive, and can be deployed directly to Netlify.

Pages Included : 

Landing Page (/) – Home page with hero and featured sections

Menu Page (/menu) – Menu items listing

Reservation (/reservation) – Reservation form section

About (/about) – About the restaurant/team

Contact Page (/contact) – Contact form

Gallery (/gallery) – Restaurant photos

Team Details (/team/:id) – Dynamic page for team member details

How to Use :

All pages are located in the src/pages/ folder.

Contact page is in src/components/ContactPage.jsx.

Replace text and images directly in the files to customize the template.

Styles are managed with Tailwind CSS and some custom CSS in src/index.css.

Components like Navbar, Footer, Hero sections, Cards, and Buttons are in src/components/.

The website is fully responsive for desktop, tablet, and mobile.
